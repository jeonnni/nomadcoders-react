# ⚡ Super Unit Converter (React)

React의 기초 개념을 활용하여 구현한 실시간 단위 변환기 프로젝트입니다.  
`Minutes`와 `Hours`, `Km`와 `Miles` 등 서로 다른 단위 간의 양방향 변환 기능을 제공하며, 사용자 경험을 고려한 UI를 적용했습니다.

## ✨ Features

- 🔄 **양방향 단위 변환**: Minutes ↔ Hours, Km ↔ Miles 실시간 연동
- 🔀 **Invert 기능**: 클릭 한 번으로 입력/출력 단위 전환
- 🧹 **Reset 기능**: 모든 입력값 초기화
- 🎨 **커스텀 디자인**: 깔끔한 UI
- 🔡 **Font Pairing**: Space Grotesk 폰트 적용

## 🛠 Tech Stack

- **React Hooks (`useState`)**: 컴포넌트 내에서 입력값, 변환 상태, 선택된 메뉴 등의 상태를 관리하고 UI를 동적으로 업데이트함.
- **HTML5 / CSS3** (flex, transition 등)
- **Babel** (JSX Transpiling)
- **Google Fonts** (Space Grotesk)

## 🎓 Learning Points

- **React State Management**: `useState`를 활용한 데이터 바인딩 및 상태 관리 이해
- **Component Logic**: 단위 변환 수식을 로직화하고 조건부 렌더링 적용
- **CSS Customizing**: `appearance: none`을 활용한 Select 박스 커스텀
- **Deployment**: GitHub Pages를 이용한 프로젝트 배포 경험
